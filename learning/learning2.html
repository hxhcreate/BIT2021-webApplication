Introduction to Back End Technology

Back end development is also called server-side development. Servers that store websites, application data, and intermediate media between back-end servers and browsers or applications belong to the back-end. Simply put, everything you can't see on the screen of an application or website is the back end of the front end.

PHP

Personal homepage is a server-side HTML script / programming language. It is a simple, object-oriented, interpretive, robust, secure, high-performance, architecture independent, portable and dynamic script language. It is a multi-purpose scripting language widely used in open source, especially suitable for web development and can be embedded in HTML. Its syntax is close to C, Java and Perl and easy to learn. This language allows web developers to write dynamically generated web pages quickly.

Database

Database is a data warehouse. It is a collection of organized and shareable data stored in the computer for a long time. Databases are usually divided into three types: hierarchical database, network database and relational database. Different databases are linked and organized according to different data structures. In today's Internet, the most common database models are SQL and NoSQL.

JSP

The full name of the JSP is java server pages. It is a technology for developing dynamic web applications released by Sun company in June 1996. JSP is a web development technology based on java servlet. Its essence is a simplified servlet design. It inserts Java program segments (scriptlets) and JSP tags (tags) into the traditional web page HTML (a subset of the standard General Markup Language (. HTM,. HTML) file to form a JSP file with the suffix (*. JSP). The web application developed with JSP is cross platform, which can run not only under Linux, but also on other operating systems.

ASP.Net

ASP. Net is a part of. Net framework. It is a Microsoft technology. It is a server-side script technology that enables scripts embedded in web pages to be executed by Internet servers. It can dynamically create documents on Web servers when requesting documents through HTTP. Refers to the active server pages (dynamic server pages), a program running in IIS (Internet information server service, a web server developed by windows).
