The core technology of front-end development

We know that the so-called web three swordsmen can no longer meet the needs. What technology should front-end development learn? The web page is mainly composed of three parts: structure, performance and behavior. The new standard of web page is W3C, and the current mode is HTML, CSS and JavaScript.

What is HTML?

HTML page is what we often call web page, also known as HTML document. HTML documents contain HTML tags and text content. The browser will read the HTML documents and present them to users in the form of web pages. HTML pages can contain pictures, text, sound, video, animation, hyperlinks, and so on. In Web documents, tags are the basic units of web pages, also known as tags and elements. They are keywords surrounded by angle brackets. HTML tags are divided into single tags and double tags. Single tags are directly closed in the start tag, while double tags are composed of start tags and end tags, and the content is filled in the middle of the tag, Most HTML tags can add attributes to decorate the tag.

What is CSS?

CSS is a language that defines style structure, such as font, color, location, etc. it is used to describe the formatting and realistic way of information on Web pages. CSS styles can be stored directly in HTML web pages or separate style sheet files. Either way, a style sheet contains rules for applying styles to elements of a specified type. When used externally, style sheet rules are placed in a file with a file extension_ CSS external stylesheet document.

Style rules are formatting instructions that can be applied to elements in a web page, such as text paragraphs or links. A style rule consists of one or more style attributes and their values. The internal style sheet is directly placed in the web page, and the external style sheet is saved in a separate document. The web page links the external style sheet through a special label.

What is JavaScript

JavaScript (JavaScript) is an object-based and event driven scripting language with security performance. Using JavaScript can easily realize interoperability with HTML and complete rich page interaction effects. It is implemented in the standard HTML language by embedding or calling in. Its emergence makes up for the defects of HTML, It is a compromise between Java and HTML.
